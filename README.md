## [Projet 12 : Détectez des faux billets avec Python](<./P12 - Détectez des faux billets avec R ou Python>)
**🎓 Compétences acquises**
* Utiliser un modèle d'apprentissage supervisé pour réaliser une prédiction (Régression logistique, kNN, Random Forest).
* Entraîner un modèle d'apprentissage non supervisé (K-means).
* Évaluer les performances des modèles et choisir le plus efficace.
* Construire une application prédictive fonctionnelle.

**📋 Scénario**
En tant que consultant Data Analyst, j'effectue une mission pour l’Organisation nationale de lutte contre le faux-monnayage (ONCFM). L'objectif est de mettre à disposition des équipes une application de Machine Learning capable de détecter la contrefaçon à partir des dimensions géométriques des billets.

**🎯 Objectifs**
* Explorer, préparer et nettoyer un jeu de données réel de 1 500 billets scannés.
* Construire et comparer 4 algorithmes de classification (K-means, Régression logistique, KNN, Random Forest).
* Isoler le meilleur algorithme dans un script autonome de prédiction en production.

## [Projet 11 : Produisez une étude de marché avec Python](<./P11 - Produisez une étude de marché avec R ou Python>)
**🎓 Compétences acquises**
* Réduire la dimension du jeu de données pour optimiser le temps d'apprentissage (ACP).
* Exploiter un modèle d'apprentissage non supervisé (Clustering CAH et K-Means).
* Réaliser des analyses multivariées pour comprendre la corrélation entre les variables.
* Sélectionner les variables pertinentes d'un modèle d'apprentissage.

**📋 Scénario**
En poste chez *La poule qui chante*, une entreprise française d’agroalimentaire, je suis chargé d'identifier les meilleurs pays pour une expansion à l'international, en ciblant des groupements de pays pertinents.

**🎯 Objectifs**
* Extraire et nettoyer les données de la FAO (Food and Agriculture Organization) et identifier des variables supplémentaires via une analyse PESTEL.
* Réaliser une Analyse en Composantes Principales (ACP) avec analyse du cercle des corrélations.
* Effectuer une Classification Ascendante Hiérarchique (CAH) suivie d'un algorithme K-Means pour créer des clusters de pays stratégiques.
